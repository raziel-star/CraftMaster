# Blocky123 Bot

This is a bot designed to automate various tasks, likely within a game environment, leveraging a modular architecture for different functionalities.

## Features

The bot's capabilities are organized into several modules:

-   **Bring:** Functionality related to bringing items or entities.
-   **Builder:** Tools for automated construction or building tasks.
-   **Chat:** Manages in-game chat interactions.
-   **Collector:** Automates the collection of resources or items.
-   **Combat:** Provides automated combat assistance.
-   **Crafter:** Handles automated crafting processes.
-   **Farmer:** Automates farming activities.
-   **Pathfinder:** Implements pathfinding algorithms for navigation.
-   **Utils:** A collection of utility functions used across other modules.

## Installation

To get started with Blocky123 Bot, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/raziel-star/Blocky123.git
    cd Blocky123
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
